Documentation sur l’application Info-Dexer [Readme] 
 
Prérequis :  
- Un ordinateur Windows ou Mac. 
- Le JAVA JRE ou JAVA JDK (Version 12+).  
- Veuillez prévoir au moins 125MB  pour l’installation du Java JRE ou Veuillez prévoir au moins 275MB  pour l’installation du Java JDK 
- Veuillez aussi prévoir au moins au moins 100MB pour les composantes de l’application Info-Dexer (code source, documents de base de l’application). 
 
 
Installation 
- Copier le fichier Info_Dexer.zip, contenant le fichier .jar permettant de lancer l’application et de générer la structure de départ du projet et les documents de base. 
Veuillez idéalement établir un répertoire stable et fixe de votre machine à partir duquel vous lancerez l’application. 

	 
Exécution du logiciel 
-	Lancer l’application dans un terminal grâce à la ligne de commande  (2 choix):  
	o	Lancer l’application en cliquant sur le fichier jar (en cliquant sur l’icône); ou 
	o	Dans le terminal exécuter : $ java -jar {nomduFichier}.jar 
 
Fonctionnalités 
- Au premier lancement de l’application, des documents de base seront copiés afin de permettre une utilisation rapide de l’application sans intervention initiale. 

- Vous pouvez effectuer des recherches par mots clés qui feront l’usage d’un moteur de recherche qui lui puisera ses correspondances à même un index généré grâce au document présent dans le répertoire « activeDocs ».

- Pour ajouter ou retirer des documents de l’index, veuillez utiliser le menu « Index » situé en haut à gauche de l’interface. Celui-ci, permet d’ajouter et de retirer de documents de la base documentaire « activeDocs » et de par la suite enclencher des changements à l’index. 
 

Important : Nous prenons pour acquis que vous aurez configuré votre environnent de travail au préalable (logiciels requis, variables d'environnement, etc.).
